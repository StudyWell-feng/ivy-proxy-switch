# Ivy Proxy Switch

一款优雅的浏览器代理切换插件，支持多代理配置管理与域名规则分流。

## 功能特性

- 🚀 **一键切换** - 弹出面板快速切换代理
- 🎨 **多配置管理** - 添加、编辑、删除多个代理配置，带颜色标签
- 🔀 **规则分流** - 按域名/URL 模式自动选择代理（PAC 模式）
- 🌐 **多协议支持** - HTTP / HTTPS / SOCKS4 / SOCKS5
- 💻 **系统代理** - 一键切回系统代理
- 🚫 **绕过规则** - 配置不走代理的地址（支持通配符）
- 🔔 **状态徽章** - 工具栏图标实时显示当前代理状态

## 安装方法（开发者模式）

1. 打开 Chrome，地址栏输入 `chrome://extensions/`
2. 开启右上角 **"开发者模式"**
3. 点击 **"加载已解压的扩展程序"**
4. 选择本项目文件夹 `ivy-proxy-switch`
5. 插件图标出现在工具栏 ✅

## 使用说明

### 添加代理

1. 点击工具栏图标打开面板
2. 点击 **"添加代理配置"**
3. 填写名称、选择协议、输入服务器地址和端口
4. 可选：设置不走代理的地址
5. 点击保存

### 切换代理

直接点击列表中的代理配置即可激活，当前激活项显示紫色高亮和「当前」徽章。

### 规则分流

1. 点击顶部 **☰ 规则** 按钮
2. 添加匹配规则（域名或 URL 通配符）
3. 为每条规则指定使用的代理
4. 设置「默认（其余流量）」的代理
5. 返回主页，激活任意代理配置即可让规则生效

> 💡 规则分流基于 PAC 脚本，适合需要「国内直连、海外走代理」的场景
